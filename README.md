# Creating a PR
```
git clone https://github.com/TsarSPb/hello-world.git
cd .\hello-world\
git branch -a
> * master
>   remotes/origin/HEAD -> origin/master
>   remotes/origin/master
git checkout -b first-pr
> Switched to a new branch 'first-pr'
git commit -a -m "Add some details"
```
